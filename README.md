# Customer Segmentation Project

## Overview
This project performs customer segmentation using K-Means clustering.

## Features
- Customer behavior analysis
- Purchase pattern segmentation
- Cluster visualization
- Business insights generation

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Output
Customers are grouped into different segments based on income and spending behavior.
